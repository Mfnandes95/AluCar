AluCar - Sistema de Agendamento e Locação de Veículos

O AluCar é um sistema completo para gerenciamento de locação de veículos, permitindo agendamentos, reservas, devoluções e inspeções veiculares. 
Desenvolvido para otimizar o fluxo de locadoras e garantir maior controle sobre a frota.

Requisitos Funcionais

Módulo de Usuário

| Funcionalidade | Descrição |
| --- | --- |
| Cadastro | Registrar nome, CPF, idade, RG, CNH, e-mail e senha |
| Login | Autenticar com e-mail e senha |
| Visualizar histórico | Consultar todas as locações realizadas |

Módulo de Agendamento e Reserva

| Funcionalidade | Descrição |
| --- | --- |
| Agendar Veículo | Escolher data e horário para retirada |
| Verificar disponibilidade | Validar se o veículo está livre no período |
| Consultar Veículos | Buscar veículos disponíveis por categoria ou modelo |
| Reservar Veículo | Confirmar a locação do veículo escolhido |
| Cancelar Agendamento | Desmarcar uma reserva antes da retirada |

Módulo de Devolução

| Funcionalidade | Descrição |
| --- | --- |
| Registrar Devolução | Iniciar o processo de devolução do veículo |
| Entregar Veículo | Usuário entrega o veículo na locadora |
| Revistar Veículo | Mecânico inspeciona o veículo (danos, combustível, quilometragem) |
| Reportar Problemas | Registrar avarias ou defeitos encontrados |

Módulo do Mecânico

| Funcionalidade | Descrição |
| --- | --- |
| Revistar Veículo | Realizar vistoria técnica detalhada |
| Registrar Problemas | Documentar irregularidades no sistema |
| Liberar Veículo | Aprovar o veículo para nova locação |
