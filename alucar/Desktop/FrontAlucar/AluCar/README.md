# AluCar - Landing Page

Landing page moderna e responsiva para a locadora de veículos AluCar, desenvolvida com Vite, HTML5, Tailwind CSS e JavaScript.

# 🚀 Tecnologias Utilizadas

- HTML5 / CSS3 / JavaScript
- Tailwind CSS
- Vite

## 🛠️ Como rodar o projeto localmente

```bash
# Instale as dependências
npm install

# Volte uma pasta
cd AluCar

# Inicie o servidor de desenvolvimento
npm run dev
